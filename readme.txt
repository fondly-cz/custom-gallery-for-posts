# Custom Gallery for Posts

Custom Gallery for Posts is a WordPress plugin that allows you to create and display custom galleries within your posts using Elementor's dynamic tags.

## Features

- Easy to use: Simply install and activate the plugin, and you're ready to start creating custom galleries using Elementor's dynamic tags.
- Customizable: Customize the appearance and layout of your galleries to match your website's design.
- Responsive: The galleries created with this plugin are fully responsive, ensuring a seamless user experience on all devices.
- Lightbox integration: Enable lightbox functionality to showcase your images in a sleek and interactive way.
- Shortcode support: Use the provided shortcode to easily embed your custom galleries into any post or page.

## Installation

1. Download the plugin zip file.
2. Log in to your WordPress admin panel.
3. Go to "Plugins" > "Add New".
4. Click on the "Upload Plugin" button.
5. Choose the plugin zip file and click "Install Now".
6. Once the installation is complete, click "Activate Plugin".

## Usage

1. After activating the plugin, you will see a new menu item called "Custom Gallery" in your WordPress admin panel.
2. Click on "Custom Gallery" to create a new gallery.
3. Add images to your gallery by uploading them or selecting from the media library.
4. Customize the gallery settings, such as layout, image size, and lightbox options.
5. Copy the provided shortcode and paste it into your post or page using Elementor's dynamic tags to display the gallery.
